## modes


### Signature

`<modes></modes>`


| Parameter | Description |
| --- | --- |
| str | SNAPSHOT – HTTP snapshot |
| str | MJPEG– HTTP motion jpeg |
| str | H264 – RTSP/(UDP)RTP H.264  |
| str | H264I – RTSP/(TCP)RTP Interlaced H.264 |
| str | H265 - HEVC |

### Example

```xml
<capabilities>
    <modes>MJPEG</modes>
</capabilities>
```
