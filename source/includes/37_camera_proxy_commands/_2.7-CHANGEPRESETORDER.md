## CHANGE\_PRESET\_ORDER

Change the order of the Presets


### Name

`CHANGE_PRESET_ORDER ()`


| Parameter | Description                                                                            |
| --------- | -------------------------------------------------------------------------------------- |
| STR       | Presets: Comma separated list of preset IDs in the new order. For example: `"4,5,2,1"` |


### Returns

`None`
