## MOVE\_TO

Have the camera move to location x and y within the specified window.


### Name

`MOVE_TO ()`


| Parameter | Description                |
| --------- | -------------------------- |
| int       | Width                      |
| int       | Height                     |
| int       | The desired  x-axis value. |
| int       | The desired  y-axis value. |


### Returns

`None`
