## USE\_HTTPS\_CHANGED

Used to update the proxy to tell it to generate HTTPS URLs instead of HTTP URLs.


### Name

`USE_HTTPS_CHANGED ()`


| Parameter | Type | Description                                                                    |
| --------- | ---- | ------------------------------------------------------------------------------ |
| USE HTTPS | BOOL | A flag that indicates an HTTPS URL should be generated instead of an HTTP URL. |


### Returns

`None`
